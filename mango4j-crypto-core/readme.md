## mango4j-hibernate-proxy-resolver
> Bitstep's common crypto support

### How to use?

#### Maven
It can be used by projects by adding the following dependency to the POM file
~~~xml
    <dependency>
        <groupId>ie.bitstep.mango</groupId>
        <artifactId>mango4j-crypto-core</artifactId>
        <version>${mango4j-utils.version}</version>
    </dependency>
~~~
