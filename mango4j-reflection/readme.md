## mango4j-reflection
> Bitstep's common utilities

Reflection classes for Java applications.

### How to use?
#### Maven
It can be used by projects by adding the following dependency to the POM file
~~~xml
    <dependency>
        <groupId>ie.bitstep.mango</groupId>
        <artifactId>mango4j-reflection</artifactId>
        <version>${mango4j-reflection.version}</version>
    </dependency>
~~~
